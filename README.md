# Metall Meister // Fusion 360 to grbl

**確実に削り出すための、最小限のパス設定ガイド。**
*The minimalist workflow for reliable toolpath generation from Fusion 360 to grbl.*

---

## 📽 Overview
このリポジトリは、YouTube解説動画 **「Metall Meister // Fusion 360 to grbl：確実に削り出す最小パス設定」** で使用した学習用データと設定資料を公開しています。

初心者でも迷わず、最短ルートで実機加工（CNC）に到達することを目的に構成されています。


---

## 📦 Contents
本リポジトリには以下のファイルが含まれています。

- **`ギアサンプル.step`**: 動画で使用しているFusion 360用の3Dモデルデータ。

---

## 🛠 Key Workflow
本ガイドで解説している「Metall Meister式」の3つのポイント：

1. **Setup Optimizer**: grbl環境におけるワーク座標系の正しい設定。
2. **Minimal Path Selection**: 汎用性が高く、失敗しにくい工具経路の選択。
3. **Reliable G-code**: 確実に動作させるためのポスト処理の手順。
